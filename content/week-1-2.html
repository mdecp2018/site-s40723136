week1-2網站更改

使用可攜內部附帶的(python)打開wsgi.py這個檔案

再用瀏覽器開啟近端

更改預設代碼 admin

若直接做更改原始碼可以參考下面列出幾項 :

<h1>會將後續文字改成大標，其h2、h3擇依次

<p>呈現的內文

<a>超連結

<iframe>插入特定視窗

若更改完成按下[save]儲存

接著按下[generate_pages]傳成靜態

再來在黑窗 打上

git add .

git commit -m "commit description"

git push

接著再輸入帳號密碼就能完成這次的更改
